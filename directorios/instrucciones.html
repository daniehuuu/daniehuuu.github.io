Instrucciones para el Algoritmo de Flujo Máximo

Pasos para interactuar con el grafo:

  • Haz doble clic en el lienzo para agregar un nodo.
  • Haz clic derecho en un nodo para ver opciones como "Cambiar etiqueta", "Conectar nodo" o "Eliminar nodo".
  • Para conectar dos nodos, primero haz click derecho en el nodo de origen y en "Conectar nodo", y luego lo mismo con el nodo de destino.
  • Para colocar una capacidad infinita en una arista, escribe "inf", "infty" o "infinity".
  • Haz clic derecho en una arista para editar su etiqueta o eliminarla.
  • Usa los botones para cargar diferentes demos de grafos o limpiar el grafo.
  • Selecciona un nodo como fuente o sumidero desde el menú contextual.

Botones de acción:

Utiliza los botones disponibles para ejecutar las siguientes acciones:

  • Iniciar Algoritmo: Ejecuta el algoritmo de flujo máximo una vez que hayas definido los nodos fuente y sumidero.
  • Limpiar Grafo: Borra todos los nodos y aristas del lienzo.
  • Demos: Carga diferentes ejemplos de grafos predefinidos para pruebas rápidas.
  • Para mostrar el detalle del algoritmo, coloque F12 y seleccione la consola.
